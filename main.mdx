<Layout>

## React Native basically...
<ul>
    <Appear>
        <li>combines native elements with React</li>
        <li>have core modules for bridging native functionality in JSX</li>
        <li>is cross-platform</li>
        <li>can be integrated with existing native code</li>
    </Appear>
</ul>

</Layout>

---

<a href="https://snack.expo.io/@freybourne/react-native:-a-tutorial" target="blank">GRAND DEMO</a>

---

<Layout>

## Pros of using React Native
<ul>
    <Appear>
        <li>one tech stack, all platforms</li>
        <li>minimal adjustments needed for React developers</li>
        <li>instantly see your changes</li>
        <li>live debugging</li>
        <li>it takes full advantage of ReactJS</li>
        <li>independent, can be combined with native</li>
        <li>frameworks built on top like <b>expo</b> makes it easier</li>
    </Appear>
</ul>

</Layout>

--- 

<Layout>

## Cons of using React Native
<ul>
    <Appear>
        <li>loses to native in all metrics</li>
        <li>developer might end up needing more than 3 skills</li>
        <li>3 mobile environments burn out</li>
        <li>increased complexity when combined with native</li>
        <li>javascript is single-threaded</li>
        <li>lacking core modules on some phone functionality like bluetooth</li>
    </Appear>
</ul>

</Layout>

## Deploying your React Native Application

---

<Layout>

Update app.json

<Smaller>

```
{
   "expo": {
    "name": "Your App Name",
    "icon": "./path/to/your/app-icon.png",
    "version": "1.0.0",
    "slug": "your-app-slug",
    "sdkVersion": "XX.0.0",
    "ios": {
      "bundleIdentifier": "com.yourcompany.yourappname"
    },
    "android": {
      "package": "com.yourcompany.yourappname"
    }
   }
 }
```

</Smaller>

</Layout>

---

<Layout> 

### Using Expo CLI

`expo build:android -t app-bundle`<br/>
`expo build:android -t apk`

<Appear>

- Upload generated apk/bundle to the play store console. 

</Appear>

</Layout>

<Layout>

### Using Expo CLI

`expo build:ios`
<ul>
    <Appear>
        <li>Enter Apple Developer Credentials</li>
        <li>Choose what team</li>
        <li>Choose a distribution certificate, or let expo handle it</li>
        <li>Upload .ipa file through XCode Application Loader</li>
        <li>Go through the app store review process<br/>through developer portal</li>
    </Appear>
</ul>
